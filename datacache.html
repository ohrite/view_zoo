DataCache Options

Option Default Description
indexer required A function that takes the key and value (which will be null for items which have not yet been looked up) and returns a string of the searchable terms in the item.
batchSize 100 Maximum number of items to look up with each ajax call.
url null URL to use for ajax lookup calls.
cache {} Can be used to pre-seed loaded items. Generally not recommended.
queryRequestFormatter null A function takes an array of keys and returns a value which can be used as the data parameter to the ajax url.
queryResponseFormatter (passthrough) A function that takes the ajax response and turns it into a map of keys and values.
ajaxType "post" HTTP method of the ajax lookup.
ajaxDelayMs 50 A throttle on the speed with which ajax lookups are made
