Options

Option Default Description
viewport required A jquery reference to the DOM element that holds the filtered list. If using AJAX to populate the data cache, this element must have an id or selector must be set.
dataCache required The DataCache object which stores the items.
renderHit required A function which creates HTML for an item that has full data; receives the key and the cache data for the item.
renderMiss required A function which creates HTML for an item that has not fetched data yet; receives the key of the item. Note that this is required even in the case of a pre-populated cache (i.e. static data set) because it is used to size the elements.
filterBox null A jquery reference to the DOM input that holds the search terms
keys [] An ordered list of the items that should be displayed.
scrollable false Set to true to allow the user to scroll the filtered list
selectable false Set to true to allow users to click on items. Selected items are remembered even when filtered out by the search terms.
selector viewport.attr('id') A string that jquery can use to find the DOM element that contains the filtered list; the data cache will use this to notify the filtered list of updated items.
